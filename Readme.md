Welcome to ModBus Ant!

ModBus Ant is simple open source cross platform ModBus RTU and ModBus TCP slave device testing terminal.

How to compile

1. Install .Net 9 SDK
2. Goto modbus_ant folder
3. Use comand line and run "dotnet run"
